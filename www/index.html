Welcome

This is an open-source file hosting server that adjusts images in real time

Resize

You can resize the image by adjusting width and height, while you can also resize it by retaining the aspect ratio.

Original (300x200)
Limit width: 150
Limit height: 150
150x150

Blur

Gaussian blurred image supports to be resized as well.

Default (level 5)
Level 10 (w150)
Level 15 (h150)
Level 20 (150x150)

Crop

You can crop the image based on horizontal/vertical coordinates and width/height. It supports to be resized and blurred as well.

h: 100
h: 100, y: 100
h: 100, w: 100, y: 50, x: 50
